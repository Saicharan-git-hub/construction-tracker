# 🏗️ Construction Project Progress Tracker

## 📌 Overview

The Construction Project Progress Tracker is a full-stack MERN web application designed to manage construction projects efficiently. It allows users to track tasks, monitor budgets, and visualize project progress in real time.

---

## 🚀 Live Demo

Frontend: https://construction-tracker-delta.vercel.app
Backend: https://your-backend.onrender.com

---

## 🛠️ Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* Authentication: JWT
* Deployment: Vercel (Frontend), Render (Backend)

---

## ✨ Features

* User Authentication (Login/Register)
* Role-Based Access (Manager, Engineer, Worker)
* Project Management (Create, Update, Delete)
* Task Tracking with Progress Updates
* Budget & Expense Monitoring
* Interactive Dashboard with Charts
* PDF Report Generation
* Alerts & Notifications

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Saicharan-git-hub/construction-tracker.git
cd construction-tracker
```

### Backend setup

```bash
cd backend
npm install
npm run dev
```

### Frontend setup

```bash
cd frontend
npm install
npm run dev
```
